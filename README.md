# ⚽ 足球看场 — AI驱动的竞彩赛事分析工具

单文件HTML应用，零依赖，直接在浏览器中打开即可使用。

## 功能

- 📊 **比赛数据** — 6大联赛40+场次，CDN实时数据+离线缓存
- 🤖 **AI预测** — 8因子加权模型（状态、主客场、进球趋势、赔率、交锋、排名、伤停、体能）
- 📈 **回测验证** — 预测准确率追踪、按联赛/方案/星级分解
- 💰 **投注系统** — 虚拟余额、投注弹窗、自动结算、盈亏追踪
- 🔧 **工具箱** — 模拟器、攻防沙盘(泊松)、凯利实验室、因子拆解、串关分析
- 📱 **PWA** — 可安装到桌面、暗/亮主题切换
- 📤 **Canvas分享** — 精美卡片图片一键下载

## 使用

1. 下载 `index.html`
2. 双击在浏览器中打开
3. 勾选免责声明即可使用
